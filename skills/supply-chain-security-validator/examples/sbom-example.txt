# SBOM Generation Example: Node.js Application

## Input
artifact_type: source
artifact_location: /path/to/nodejs-app
sbom_format: both
verification_tier: T1

## Commands
# Generate SPDX SBOM
syft dir:/path/to/nodejs-app -o spdx-json=app-sbom.spdx.json

# Generate CycloneDX SBOM
syft dir:/path/to/nodejs-app -o cyclonedx-json=app-sbom.cdx.json

# Scan for vulnerabilities
grype sbom:app-sbom.spdx.json --fail-on medium
